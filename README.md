## Chức năng từng đối tượng
### Quản trị viên
- Đăng nhập
- Quản lý trang thông tin: logo, banner, giới thiệu,…
- Quản lý người dùng: thông tin cá nhân người dùng, nhân viên
- Quản lý xe
    + Tên xe
    + Ảnh
    + Hãng
    + Chỗ ngồi
    + Địa điểm
    + Mô tả
    + Điều khoản
    + Mức giá niêm yết
    + Mức giá 1 ngày
    + Đánh giá
- Quản lý hóa đơn
    + id
    + id người dùng
    + id xe
    + Ngày bắt đầu
    + Ngày kết thúc
    + Tổng tiền
    + Trạng thái: đặt xe, giao xe, quá hạn trả xe, đã thanh toán

### Người dùng
- Đăng nhập, đăng kí
- Quản lý thông tin cá nhân
    + Tên
    + Giới tính
    + email 
    + địa chỉ
    + tài khoản
    + mật khẩu
- Quản lý hóa đơn
    + Chi tiết hóa đơn
    + Lịch sử thuê xe
